# FloorPlan
Furniture Recommendation using 2D Floorplan

# Installation
```
git clone https://github.com/Zeppelinpp/FloorPlan.git
cd FloorPlan
```
**Prepare dependencies**
```
pip install uv # If uv is uninstalled
```
**Sync dependencies**
```
uv syncsfsdf s
```
